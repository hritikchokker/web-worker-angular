A guide to use web worker in angular app

In this sample i will demonstrate the usage of web workers in javascript and how they can help you in the case when you have many complex and time taking tasks to perform on your frontend.

Web Worker helps to offload the task to worker thread. this will save the main thread from running a complex task.
This process will be explained using a fibonnaci program.
Here I have two sample input which will find the fibonnaci number using a recursive algorithm. one sample will run on the same thread where our present code is executing and the other sample will run on a separate thread.
I Have added some dummy checkbox,range,and radio inputs to show that if we run a complex task on main thread everything will freeze.
